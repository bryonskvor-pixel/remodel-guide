# remodel-guide# remodel.guide

The Homeowner Protection Guide for Residential Remodeling.

A free educational resource built on 20+ years of residential 
construction experience. Designed to help homeowners understand 
every major remodeling project before a contractor shows up.

---

## What this is

remodel.guide is a companion site to remodelry.guide — 
Remodelry's remodeling concierge service serving Northeast Ohio.

Every page covers one project type completely:
- Full scope
- Real timeline
- Every decision in order
- Contractor checklist
- What to watch for
- Questions to ask

Every page ends with a call to action to Talk to Remi at 
remodelry.guide/remi.
